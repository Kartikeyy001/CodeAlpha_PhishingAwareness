# Phishing Awareness Training - CodeAlpha Internship

## Overview
Comprehensive online phishing awareness training module covering:
- How to recognize phishing emails and fake websites
- Social engineering tactics and psychological exploitation
- Real-world case studies (Sony Pictures, Colonial Pipeline)
- 4-phase attack methodology
- Interactive knowledge checks with scenario-based learning

## Features
- 17 professional slides
- Dark theme (cybersecurity standard)
- 3 interactive quiz scenarios
- Real statistics and case studies
- Defense strategy framework

## How to Use
1. Open `phishing-awareness-training.html` in any web browser
2. Navigate through slides to learn about phishing awareness
3. Complete interactive quizzes to test knowledge
4. Share with colleagues and friends for their training

## Presentation Highlights
- Case Study: Sony Pictures ($100M loss)
